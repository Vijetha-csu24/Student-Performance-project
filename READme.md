# Student Performance Predictor 📚

This project uses machine learning to predict students' final grades (G3) based on behavioral and academic data from the UCI Student Performance dataset.

## 🔍 Dataset
- Source: [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/320/student+performance)
- File: `student-mat.csv`

## 🧠 Methods
- Preprocessing: One-hot encoding for categorical variables
- Model: Random Forest Regressor
- Evaluation: R² Score, Mean Squared Error
- Visualization: Actual vs Predicted, Residual plot

## 📊 Goal
To demonstrate a basic regression model for predicting educational outcomes using real-world data.

## 💻 Libraries Used
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
